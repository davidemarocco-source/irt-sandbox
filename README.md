# 4PL IRT Educational Sandbox

A minimalistic, interactive web application to teach the **Four-Parameter Logistic (4PL)** model of Item Response Theory.

## Features
- **Interactive Graph**: Visualize the Item Characteristic Curve (ICC) and Information Function (IIF).
- **Real-time Parameters**: Adjust Discrimination ($a$), Difficulty ($b$), Guessing ($c$), and Inattention ($d$) sliders.
- **Dual Views**: Toggle the Information Curve on or off.
- **Math Visualization**: Live rendering of the formula using KaTeX.
- **Zero Dependencies**: Built with Vanilla JS, HTML, and CSS (using CDNs).

## How to Run Locally
Simply open `index.html` in your web browser.

## Deployment
This project is ready for **GitHub Pages** or **Netlify Drop**.
- Just upload these files to your repository and enable generic static site hosting.
